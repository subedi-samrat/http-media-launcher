About HTTP Media Launcher

HTTP Media Launcher is a cross-platform tool that helps developers launch web browsers with the necessary flags to enable camera and microphone access on HTTP sites.


Features:

  • Support for Chrome, Brave, Firefox, Edge, Chromium, and Safari
  • Cross-platform compatibility (Windows, macOS, Linux)
  • Automatic browser detection
  • Safe temporary profile creation
  • Command export functionality

Help & Usage

How to Use:

  1. Enter the URL of your HTTP site
  2. Select your preferred browser from the grid
  3. Click "Launch Browser" to start the browser with media permissions

Browser Support:

  • Chrome/Brave/Edge: Uses Chromium flags for unsafe origins
  • Firefox: Creates temporary profile with media permissions
  • Safari: Limited options (manual setup may be required)

Security Notes:

This tool temporarily disables certain browser security features. The launched browser instance uses a separate profile to avoid affecting your regular browsing data.

Troubleshooting:

  • If a browser shows as "not found", try refreshing the browser list
  • Make sure the browser is properly installed
  • Try running the app as administrator (Windows) if browser launch fails
